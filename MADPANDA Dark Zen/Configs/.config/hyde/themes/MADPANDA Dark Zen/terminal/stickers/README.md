# Terminal Stickers

This directory is the theme-owned pool for transparent terminal cutouts.

Dark Zen uses these PNGs instead of HyDE's random terminal art while the Dark
Zen theme is active.

`mad-theme-pack apply` copies these files into the live theme directory:

`$XDG_CONFIG_HOME/hyde/themes/MADPANDA Dark Zen/terminal/stickers/`

If the active theme has no theme-pack sticker images, the zsh hook falls back
to HyDE's normal random terminal art path. It should not keep showing Dark Zen
stickers after the theme changes.
