# MADPANDA Dark Zen Source assets

This package bundles the inherited Bad Blood visual assets needed by fresh HyDE
installs:

- GTK theme alias: MADPANDA-Dark-Zen
- Icon theme: besgnulinux-mono-red
- Cursor theme: Night-Diamond-Red

The cursor archive intentionally preserves the upstream Night-Diamond-Red
payload. Be careful when changing hyprcursor manifest files because the
upstream Bad Blood source notes that enabling hyprcursor manifests can crash
Hyprland on some systems.
